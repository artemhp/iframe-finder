# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-31

### Added
- Initial release
- `findIframeById()` - Find iframe by id attribute
- `findIframeByName()` - Find iframe by name attribute
- `findIframe()` - Find iframe with custom predicate
- `findAllIframes()` - Find all matching iframes
- Recursive search through nested iframes
- Cross-origin iframe safety handling
- TypeScript type definitions
- Comprehensive documentation
